# angular-gref

it's not ```href```. it's ```gref```.

google reference for images

#### Step 1: Put img tags in ```<gref>```.
```html
   
  <gref>
	  <img alt="Pulp Fiction" src="http://tiny.cc/ahgx7x">
  </gref>
   
```

#### Step 2: And then, click image and google it image.

#### Step 3: Close.
